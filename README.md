## Instructions for Local Execution

To run this project successfully, you will need to execute the training script on your local machine with sufficient resources.

**1. Setup Environment:**

*   Ensure you have Python 3.9+ installed.
*   Navigate to the project directory (`toxic-comment-classifier`).
*   It is highly recommended to create a virtual environment:
    ```bash
    python -m venv venv
    source venv/bin/activate  # On Windows use `venv\Scripts\activate`
    ```
*   Install the required dependencies:
    ```bash
    pip install -r requirements.txt
    ```
    *Note: This includes PyTorch, Transformers, scikit-learn, pandas, etc. Ensure you have sufficient disk space.* 
    *If you have a CUDA-enabled GPU, PyTorch should automatically detect and use it, significantly speeding up training. Check PyTorch installation instructions for specific CUDA versions if needed.* 

**2. Train the Model:**

*   Run the training script:
    ```bash
    python train_bert.py
    ```
*   **Customization (Optional):** You can modify `train_bert.py` to adjust:
    *   `DATA_SUBSET_SIZE`: Set to `None` or remove the `.head()` line to train on the full dataset (requires significant time and resources).
    *   `EPOCHS`: Increase the number of epochs for potentially better performance (default is 1 for quick testing).
    *   `BATCH_SIZE`: Adjust based on your GPU memory (default is 8).
*   This script will preprocess the data (`data/toxic_comments.csv`), train the DistilBERT model, evaluate it, and save the trained model weights to `model_bert.pth` in the project root directory.

**3. Run the Streamlit Application:**

*   Once `model_bert.pth` has been generated by the training script, you can run the interactive web application:
    ```bash
    streamlit run app.py
    ```
*   This will start a local web server, and you can access the classifier interface in your browser (usually at `http://localhost:8501`).
